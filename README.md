Task Tracker

Task Tracker — консольное приложение на Java для управления задачами и сотрудниками. Программа позволяет создавать задачи, 
назначать их исполнителям, изменять статус выполнения и получать статистику по задачам.
Все данные хранятся в памяти программы с использованием коллекций Java (`ArrayList` и `HashMap`).


Функционал
* Добавление сотрудников
* Создание задач
* Назначение задач сотрудникам
* Изменение статуса задач
* Просмотр задач сотрудника
* Просмотр задач по статусу
* Получение статистики по задачам


Структура проекта:
* Main.java — точка входа в программу
* TaskService.java — бизнес-логика приложения
* Employee.java — модель сотрудника
* Task.java — модель задачи
* TaskStatus.java — статусы задач
* Priority.java — приоритеты задач


Инструкция по запуску:
1. Открыть проект в IntelliJ IDEA.
2. Найти файл Main.java и запустить его.
3. Для использование нужного элемента пропишите цифру нужного действия (1-8) и нажмите enter.

Меню программы состоит из следующих элементов:
=== TASK TRACKER ===
1. Добавить сотрудника
2. Создать задачу
3. Назначить задачу
4. Изменить статус задачи
5. Просмотр задач сотрудника
6. Просмотр задач по статусу
7. Статистика по задачам
8. Выход
